## Tipos de Promociones Soportados

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                                                                             │
│   TIPOS DE PROMOCIONES Y DESCUENTOS                                         │
│                                                                             │
│   ┌─────────────────────────────────────────────────────────────────────┐   │
│   │                                                                     │   │
│   │   % PORCENTAJE                                                      │   │
│   │   ─────────────                                                     │   │
│   │   • Descuento X% sobre precio                                       │   │
│   │   • Aplica a: todo, categorías, productos                           │   │
│   │   • Puede tener tope máximo de descuento                            │   │
│   │                                                                     │   │
│   │   $ MONTO FIJO                                                      │   │
│   │   ────────────                                                      │   │
│   │   • Descuento de $X fijos                                           │   │
│   │   • Por producto o por ticket                                       │   │
│   │                                                                     │   │
│   │   N x M                                                             │   │
│   │   ─────                                                             │   │
│   │   • Lleva N productos, paga M                                       │   │
│   │   • 2x1, 3x2, etc.                                                  │   │
│   │   • Mismo producto o combinación                                    │   │
│   │                                                                     │   │
│   │   💰 PRECIO ESPECIAL                                                │   │
│   │   ──────────────────                                                │   │
│   │   • Precio fijo temporal                                            │   │
│   │   • Ejemplo: "Esta semana a $15"                                    │   │
│   │                                                                     │   │
│   │   🎁 REGALO / BONUS                                                 │   │
│   │   ─────────────────                                                 │   │
│   │   • Al comprar X, recibe Y gratis                                   │   │
│   │   • Producto diferente de regalo                                    │   │
│   │                                                                     │   │
│   │   📦 PAQUETE / BUNDLE                                               │   │
│   │   ───────────────────                                               │   │
│   │   • Conjunto de productos a precio especial                         │   │
│   │   • Ejemplo: "Combo refresco + papas = $35"                         │   │
│   │                                                                     │   │
│   │   📊 VOLUMEN / ESCALADO                                             │   │
│   │   ─────────────────────                                             │   │
│   │   • Descuento según cantidad comprada                               │   │
│   │   • Ejemplo: "5%>$500, 10%>$1000"                                   │   │
│   │                                                                     │   │
│   └─────────────────────────────────────────────────────────────────────┘   │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```
